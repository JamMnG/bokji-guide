# 복지길잡이

`복지길잡이`는 중앙부처 복지서비스를 검색하고 비교할 수 있는 단일 페이지 웹 애플리케이션입니다.

## 프로젝트 이름

- 저장소 이름: `bokji-guide`
- 화면 이름: `복지길잡이`

## 주요 기능

- 공공데이터포털 `data.go.kr`의 한국사회보장정보원_중앙부처복지서비스 API 연동
- 키워드 검색 및 소관부처 필터
- 생애주기 필터
- 상세 정보 모달
- 찜한 복지 저장 및 `localStorage` 유지
- 복지로 스타일의 상단 영역, 공지, 바로가기 UI

## 실행 방법

1. 프로젝트 폴더에서 환경변수를 설정합니다.
2. `node server.js`로 로컬 서버를 실행합니다.
3. 브라우저에서 `http://localhost:4173/`로 접속합니다.

```powershell
$env:DATA_GO_KR_SERVICE_KEY="공공데이터_인증키"
node server.js
```

## 파일 구조

- `index.html` : 화면 UI와 클라이언트 로직
- `server.js` : `data.go.kr` API를 호출하는 로컬 프록시 서버
- `.env.example` : 환경변수 예시

## 참고

- API 키는 저장소에 직접 넣지 않고 환경변수로 관리합니다.
- 브라우저에서 직접 파일을 열면 일부 기능이 동작하지 않을 수 있으므로, 로컬 서버 실행을 권장합니다.
- API 응답이 불안정할 때는 샘플 데이터로 대체 표시됩니다.
